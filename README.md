Farm Assist:
Farm Assist is a farmer-centric application that helps resolve crop-related queries with support for offline usage and multiple languages. It enables farmers to seek guidance in their native language and access assistance even in low-connectivity areas.
Features:
1.Crop-related query assistance
2.Multi-language support
3.Offline functionality
4.Contact section
